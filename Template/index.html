Membership

Picture Youtube, Patreon and Zoom had a baby. That's Membership. A platform where you can create paid memberships, upload content with Video On Demand services, and schedule meetings with your members. I've created the whole platform from scratch, from the UI to the backend. The platform has over 1500 active users and it's growing every day.

Project Description

The story

A finance content creator from Canada reached out as he wanted to create a platform to upload content and schedule meetings with his members. The project started as a simple platform, but as the project grew, we added more features like Video On Demand, Live Streaming, and a full membership system. The platform is built with AWS Amplify, NextJS, and GraphQL. The backend is a serverless architecture with AWS Lambda and DynamoDB.

MY APPROACH

Since the project started with an AWS Cognito authentication system, I decided to leverage AWS Amplify to create the whole platform. The frontend is built with NextJS and the backend with AWS Lambda and DynamoDB. The project is fully serverless and it's hosted on AWS Amplify. The project is still ongoing and we are adding more features every month.

FDB

Fleet Dashboard Control

This one was really interesting! So, basically what I've done is to get data about a whole fleet from telemetric devices from an API, set up an EC2 and RDS database in AWS instance and use CRON to run a Python script to run an ETL process to upload the data in a Power BI dashboard.

Project Description

The story

A customer decided to enhance their telemetry of their fleet of trucks. They wanted to have a dashboard that shows the location of the trucks, the speed, the fuel consumption, and the temperature of the trucks. They also wanted to have a report of the data of periods of time for their personnel development programs

MY APPROACH

I supply the fleet with a low-cost telemetric device that sends data to an API. I set up an EC2 and RDS instance in AWS and use CRON to run a Python script that runs an ETL process to upload the data in a Power BI dashboard.
